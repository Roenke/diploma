\begin{itemize}

	\item Intermediate operations. Промежуточные операции, которые преобразуют поток объектов.
	\begin{itemize}
		\item Без состояния
		\item С состоянием
	\end{itemize}
	\item Terminal operation. Операция которая завершает цепочку.
\end{itemize}
